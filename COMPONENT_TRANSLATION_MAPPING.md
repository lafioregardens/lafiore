## Shop.jsx

**Current hardcoded strings → New translation keys:**

| Hardcoded Text | Translation Key | Location |
|---|---|---|
| "Potted Plants" | `potPlants` | Product badge (line 327) |
| `Last {product.stock} remaining` | `lastRemaining` | Low stock badge (line 334) |
| "Sold Out" | `soldOut` | Sold out overlay (line 341) |
| `Price on request` | `priceOnRequest` | Fallback price (line 377) |

**Usage in JSX:**
```jsx
// Line 327 - Potted Plants badge
<span className="shop-card-badge shop-card-badge--potted">
  {t("potPlants")}
</span>

// Line 334 - Low stock
<span className="shop-card-badge shop-card-badge--low">
  {t("lastRemaining").replace("{count}", product.stock)}
</span>

// Line 341 - Sold out
<div className="shop-card-sold-out-tag">{t("soldOut")}</div>

// Line 377 - Price fallback
: (product.price || t("priceOnRequest"))}
```
